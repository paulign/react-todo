- React-router-dom 4.2.2

- Bootstrap 4
- Reactstrap
- Webpack 4.8.1
- Font Awesome 5

### How to start project on local machine?

 1. Open terminal in the root folder of the project;
 2. run `npm install`
 3. run `npm start`
 4. your browser will automatically open project on http://localhost:9000/
